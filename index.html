Project Description:

Flappy Bird is an application developed as part of a course project for a Java programming class. The goal of the project is to create a game that features user authentication, record tracking, and a leaderboard displaying the top players. The project also focuses on implementing the core gameplay mechanics

Technology Stack:

    1. Backend:
    • Java, Spring Boot
    • Spring JPA, Hibernate
    • Liquibase
    • MariaDB
    2. Frontend:
    • HTML, CSS(Vanilla)
    • Google Fonts
    • Font Awesome
    • JavaScript & TypeScript
    3. Testing:
    • Mockito
    • Spring DevTools

Project Architecture:

The project is built using the MVC pattern:

  • Model: Database Tables
  • View: Pages implemented using HTML and CSS
  • Controller: Request Management and Server-Client Interaction

Functionality:

    1. Registration and Authentication Page:
    • New user registration with fields for username, email, and password
    • Authentication
    • Error Display
    • Show and hide password with a button
    • Page Navigation
    2. Flappy Bird Game:
    • The basic game logic is implemented with physics elements: the bird falls down and rises when the button is pressed
    • Score Calculation
    • Saving high scores to the database
    3. User Profile:
    • Displaying user data
    • Profile Editing
    • Changing Settings
    • Resetting the score
    4. Leaderboard Page:
    • Displaying top players
    • Displaying the user's rank
    • Sorting by score
    5. Additional Pages:
    • About: Game and Developers Information
    • Menu: Main Menu for Navigation
    • Exit: Exit Game Window

Setting up and running the project:

    1. Configure the database connection file application.properties:
    spring.application.name=game
     
    # Datasource
    spring.datasource.url=*YOUR_VALUE*
    spring.datasource.username=*YOUR_VALUE*
    spring.datasource.password=*YOUR_VALUE*
     
    # JPA
    spring.jpa.hibernate.ddl-auto=update
    spring.jpa.show-sql=true
    spring.jpa.properties.hibernate.format_sql=true
     
    # Liquibase
    spring.liquibase.enabled=true
    spring.liquibase.change-log=classpath:db/changelog/db.changelog-master.xml
    spring.datasource.driver-class-name=org.mariadb.jdbc.Driver
    spring.datasource.platform=mariadb
    spring.datasource.jpa.database-platform=org.hibernate.dialect.MariaDBDialect
     2. Run the application through the main file:
    mvn spring-boot:run
     3. Open your browser and go to the address:
    http://localhost:8080
